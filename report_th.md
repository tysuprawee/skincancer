# การวิเคราะห์รูปแบบทางคลินิกของแผลผิวหนังมะเร็งเมลานอมาและไม่ใช่เมลานอมาโดยใช้ข้อมูลเมตา HAM10000

**รายงานโครงการรายวิชา**
**ชุดข้อมูล:** HAM10000 (Human Against Machine with 10,000 training images)
**ประเภทการวิเคราะห์:** การวิเคราะห์รูปแบบทางคลินิกเชิงสำรวจ ร่วมกับการทดสอบทางสถิติและการคัดกรองด้วย ML จากข้อมูลเมตา

> **ข้อจำกัดความรับผิดชอบทางวิชาการ:** โครงการนี้ดำเนินการเพื่อวัตถุประสงค์ทางการศึกษาเท่านั้น ผลการวิเคราะห์ทั้งหมดอิงจากชุดข้อมูลวิจัย และต้องไม่นำไปใช้ในบริบททางคลินิกหรือการวินิจฉัยใด ๆ

## 1. บทนำ

มะเร็งผิวหนังเป็นหนึ่งในมะเร็งที่พบได้บ่อยที่สุดในโลก ในสหรัฐอเมริกาเพียงประเทศเดียว มีการวินิจฉัยมะเร็งผิวหนังมากกว่า 5 ล้านรายต่อปี ในบรรดามะเร็งผิวหนังทุกชนิด **เมลานอมา (melanoma)** เป็นชนิดที่มีอัตราเสียชีวิตสูงสุด เป็นสาเหตุของการเสียชีวิตจากมะเร็งผิวหนังส่วนใหญ่ แม้จะคิดเป็นเพียงประมาณ 1% ของทุกกรณี อย่างไรก็ตาม เมื่อตรวจพบในระยะเริ่มต้น (ระยะ I) อัตราการรอดชีวิตห้าปีเกิน 98% แต่เมื่อตรวจพบในระยะ IV ตัวเลขนี้ลดลงเหลือประมาณ 23%

ดังนั้น การตรวจพบในระยะเริ่มต้นจึงเป็นปัจจัยสำคัญที่สุดในการปรับปรุงผลลัพธ์ของผู้ป่วย การตรวจด้วยไดเออร์โมสโกปี (dermatoscopy) เป็นเทคนิคการถ่ายภาพที่ไม่รุกราน ใช้แสงโพลาไรซ์เพื่อมองเห็นโครงสร้างผิวหนังใต้ชั้นผิว และได้กลายเป็นเครื่องมือมาตรฐานทางคลินิกในการประเมินแผลผิวหนังที่มีเม็ดสี วิธีนี้ให้ความแม่นยำในการวินิจฉัยสูงกว่าการตรวจด้วยตาเปล่าอย่างมีนัยสำคัญ

นอกจากการวิเคราะห์ภาพแล้ว **ข้อมูลเมตาทางคลินิก (clinical metadata)** เช่น อายุ เพศ และตำแหน่งของแผล ยังให้บริบททางวินิจฉัยที่มีความหมาย แผลบางชนิดมีแนวโน้มทางประชากรและกายวิภาคที่ชัดเจน ตัวอย่างเช่น เมลานอมามักพบในผู้ป่วยสูงอายุและมักปรากฏในบริเวณที่สัมผัสแสงแดด เช่น หลังและลำตัว มะเร็งเซลล์ฐาน (basal cell carcinoma) มักพัฒนาที่ใบหน้า คอ และหนังศีรษะ การเข้าใจรูปแบบเหล่านี้ช่วยให้แพทย์จัดลำดับความสำคัญว่าผู้ป่วยใดควรได้รับการประเมินอย่างใกล้ชิด

โครงการนี้ใช้ชุดข้อมูล HAM10000 เพื่อสำรวจรูปแบบทางคลินิกและประชากรศาสตร์ผ่านการวิเคราะห์ข้อมูล การทดสอบสมมติฐานทางสถิติ และการเรียนรู้ของเครื่องแบบง่าย โดยไม่พึ่งพาข้อมูลภาพ เป้าหมายคือทำความเข้าใจว่าข้อมูลเมตาอย่างเดียวสะท้อนชีววิทยาของแผลผิวหนังได้อย่างไร

## 2. คำอธิบายชุดข้อมูล

### 2.1 ภาพรวม

ชุดข้อมูล **HAM10000** (*Human Against Machine with 10,000 training images*) ตีพิมพ์โดย Tschandl และคณะ (2018) และเป็นหนึ่งในชุดข้อมูลไดเออร์โมสโกปีที่เปิดเผยต่อสาธารณะที่ใหญ่ที่สุด ประกอบด้วย **ภาพไดเออร์โมสโกปี 10,015 ภาพ** ของแผลผิวหนังที่รวบรวมจากหลายศูนย์ พร้อมข้อมูลเมตาทางคลินิกที่มีโครงสร้างสำหรับแต่ละภาพ

### 2.2 ตัวแปรข้อมูลเมตา

| คอลัมน์ | ชนิด | คำอธิบาย |
|--------|------|----------|
| `lesion_id` | สตริง | ตัวระบุเฉพาะของแผล (หนึ่งแผลอาจมีหลายภาพ) |
| `image_id` | สตริง | ตัวระบุเฉพาะของแต่ละภาพ |
| `dx` | หมวดหมู่ | รหัสการวินิจฉัย (หนึ่งใน 7 ชั้น) |
| `dx_type` | หมวดหมู่ | วิธีการยืนยันการวินิจฉัย |
| `age` | ตัวเลข | อายุผู้ป่วย (ปี) |
| `sex` | หมวดหมู่ | เพศผู้ป่วย (ชาย / หญิง / ไม่ทราบ) |
| `localization` | หมวดหมู่ | ตำแหน่งบนร่างกายที่พบแผล |

### 2.3 ชั้นการวินิจฉัย

| รหัส | ชื่อเต็ม | ลักษณะความรุนแรง |
|------|---------|-----------------|
| `nv` | ไฝเม็ดสี (Melanocytic Nevi) | ไม่ร้ายแรง |
| `bkl` | แผลคล้ายเกล็ดไม่ร้ายแรง (Benign Keratosis-like Lesions) | ไม่ร้ายแรง |
| `df` | เนื้องอกไฟโบรมาผิวหนัง (Dermatofibroma) | ไม่ร้ายแรง |
| `vasc` | แผลหลอดเลือด (Vascular Lesions) | ไม่ร้ายแรง |
| `mel` | เมลานอมา (Melanoma) | ร้ายแรง |
| `bcc` | มะเร็งเซลล์ฐาน (Basal Cell Carcinoma) | ร้ายแรง |
| `akiec` | เกล็ดจากแสงแดด / โรคโบว์น (Actinic Keratosis / Bowen's Disease) | ก่อนมะเร็งถึง in-situ |

### 2.4 วิธีการยืนยันการวินิจฉัย

| รหัส | วิธีการ |
|------|---------|
| `histo` | พยาธิวิทยา (ชิ้นเนื้อจากการส่องกล้อง มาตรฐานทองคำ) |
| `follow_up` | การติดตามทางคลินิก |
| `consensus` | ความเห็นพ้องของแพทย์ผิวหนังผู้เชี่ยวชาญ |
| `confocal` | กล้องจุลทรรศน์ confocal แบบสะท้อน |

## 3. คำถามวิจัย

### คำถามวิจัยหลัก

> อายุ เพศ และตำแหน่งบนร่างกายของผู้ป่วย มีความสัมพันธ์กับชนิดการวินิจฉัยแผลผิวหนังหรือไม่?

### คำถามย่อย

1. แผลแต่ละชนิดพบได้บ่อยที่สุดในกลุ่มอายุใด?
2. มีความแตกต่างทางสถิติที่มีนัยสำคัญในการกระจายเพศระหว่างชนิดการวินิจฉัยหรือไม่?
3. ตำแหน่งบนร่างกายบางแห่งมีสัดส่วนแผลคล้ายร้ายแรงสูงกว่าหรือไม่?
4. เมลานอมามีรูปแบบอายุและตำแหน่งที่แตกต่างจากแผลไม่ร้ายแรงหรือไม่?
5. ข้อมูลเมตาทางคลินิกอย่างเดียวสามารถให้สัญญาณคัดกรองที่มีความหมายสำหรับความเสี่ยงเมลานอมาได้หรือไม่?

## 4. วิธีการ

### 4.1 การทำความสะอาดข้อมูล

ขั้นตอนการทำความสะอาดต่อไปนี้ถูกใช้ก่อนการวิเคราะห์:

1. **ลบแถวที่เพศไม่ทราบ:** ลบ 57 แถวที่ `sex == "unknown"`
2. **ลบแถวที่ตำแหน่งไม่ทราบ:** ลบ 234 แถวที่ `localization == "unknown"`
3. **เติมค่าอายุที่ขาดหาย:** เติมค่า `age` ที่ขาดด้วยอายุมัธยฐานภายในแต่ละกลุ่มการวินิจฉัย (`dx`) เพื่อรักษาการกระจายอายุระดับกลุ่ม
4. **จัดกลุ่มอายุ:** แปลงอายุต่อเนื่องเป็นห้ากลุ่มทางคลินิก: 0-20, 21-40, 41-60, 61-80, 80+
5. **จัดกลุ่มทางคลินิก:** สร้างคอลัมน์ไบนารี `lesion_class`:
   - **ไม่ร้ายแรง:** `nv`, `bkl`, `df`, `vasc`
   - **คล้ายร้ายแรง:** `mel`, `bcc`, `akiec`

หลังการทำความสะอาด ชุดข้อมูลที่ใช้งานมีประมาณ 9,724 แถว

### 4.2 การวิเคราะห์ข้อมูลเชิงสำรวจ

สร้างภาพนิเสนอแปดภาพเพื่ออธิบายการกระจายและรูปแบบในข้อมูล:

1. จำนวนกรณีตามการวินิจฉัย (กราฟแท่งแนวนอน)
2. การกระจายอายุตามการวินิจฉัย (box plot)
3. สัดส่วนไม่ร้ายแรงและคล้ายร้ายแรงตามกลุ่มอายุ (กราฟแท่งซ้อน)
4. การกระจายเพศในแต่ละการวินิจฉัย (กราฟแท่งจัดกลุ่ม)
5. จำนวนการวินิจฉัยตาม 7 ตำแหน่งบนร่างกายหลัก (กราฟแท่งจัดกลุ่ม)
6. แผนที่ความร้อน การวินิจฉัย x ตำแหน่ง (ร้อยละภายในแต่ละการวินิจฉัย)
7. การแบ่งชั้นไม่ร้ายแรงและคล้ายร้ายแรงโดยรวม (วงกลม + แท่ง)
8. วิธียืนยันการวินิจฉัยตามการวินิจฉัย (กราฟแท่งซ้อน)

### 4.3 การวิเคราะห์ทางสถิติ

ดำเนินการทดสอบสมมติฐานสี่รายการ:

| การทดสอบ | ตัวแปร | วิธีการ |
|---------|--------|---------|
| 1 | dx vs เพศ | Pearson Chi-square |
| 2 | dx vs ตำแหน่ง | Pearson Chi-square |
| 3 | อายุ vs กลุ่ม dx | Kruskal-Wallis H test |
| 4 | อายุเมลานอมา vs ไม่ใช่เมลานอมา | Mann-Whitney U test |

ระดับความมีนัยสำคัญ: alpha = 0.05 สำหรับทุกการทดสอบ เลือก Kruskal-Wallis และ Mann-Whitney U แทน ANOVA เนื่องจากข้อมูลอายุระหว่างกลุ่มการวินิจฉัยไม่มีการแจกแจงแบบปกติ

### 4.4 การเรียนรู้ของเครื่อง (เฉพาะข้อมูลเมตา)

กำหนดงานจำแนกไบนารีเพื่อทำนาย **เมลานอมา vs ไม่ใช่เมลานอมา** โดยใช้เพียงสามคุณลักษณะข้อมูลเมตาทางคลินิก: อายุ เพศ และตำแหน่ง

- **แบ่ง train/test:** 80/20 แบ่งชั้นตามคลาส
- **การประมวลผลล่วงหน้า:** StandardScaler สำหรับอายุ; OneHotEncoder สำหรับเพศและตำแหน่ง
- **โมเดล:** Logistic Regression, Decision Tree (ความลึกสูงสุด 5), Random Forest (100 ต้น)
- **การจัดการความไม่สมดุลของชั้น:** ใช้ `class_weight='balanced'` กับทุกโมเดล
- **ตัวชี้วัดหลัก:** Recall สำหรับชั้นเมลานอมา เนื่องจากการพลาดตรวจเมลานอมามีต้นทุนทางคลินิกสูงกว่าการแจ้งเตือนผิดพลาดมาก

### 5.3 รูปแบบตามตำแหน่งบนร่างกาย

### 5.5 ผลการวิเคราะห์ทางสถิติ

**การทดสอบที่ 1: dx vs เพศ (Chi-square)**
การทดสอบ Chi-square ให้ผลที่มีนัยสำคัญทางสถิติ (p < 0.05) แสดงว่าชนิดแผลมีความสัมพันธ์อย่างมีนัยสำคัญกับเพศผู้ป่วย ความสัมพันธ์นี้ขับเคลื่อนหลักโดยเนื้องอกไฟโบรมาผิวหนัง (พบในหญิงมากกว่า) และแผลหลอดเลือด (พบในชายมากกว่า)

**การทดสอบที่ 2: dx vs ตำแหน่ง (Chi-square)**
พบความสัมพันธ์ที่มีนัยสำคัญสูงระหว่างชนิดการวินิจฉัยและตำแหน่งบนร่างกาย (p < 0.001) ยืนยันว่าแผลแต่ละชนิดมีแนวโน้มทางกายวิภาคที่แตกต่างกัน สอดคล้องกับพยาธิสรีรวิทยาที่ทราบแล้ว

**การทดสอบที่ 3: อายุตามกลุ่ม dx (Kruskal-Wallis)**
อายุมัธยฐานของผู้ป่วยแตกต่างกันอย่างมีนัยสำคัญระหว่างกลุ่มการวินิจฉัย (p < 0.001) กลุ่มแผลคล้ายร้ายแรงมีอายุมัธยฐานสูงกว่ากลุ่มไม่ร้ายแรงอย่างสม่ำเสมอ สนับสนุนบทบาทของอายุเป็นปัจจัยเสี่ยง

**การทดสอบที่ 4: อายุเมลานอมา vs ไม่ใช่เมลานอมา (Mann-Whitney U)**
ผู้ป่วยเมลานอมามีอายุมากกว่าผู้ป่วยที่ไม่ใช่เมลานอมาอย่างมีนัยสำคัญ (p < 0.001) ผลลัพธ์นี้เสริมแนวทางทางคลินิกให้ตรวจสอบแผลที่มีเม็ดสีในผู้ป่วยสูงอายุอย่างใกล้ชิด

### 5.6 ผลการเรียนรู้ของเครื่อง

โมเดลทั้งสามถูกฝึกด้วยข้อมูลเมตาอย่างเดียว (อายุ เพศ ตำแหน่ง) เพื่อจำแนกเมลานอมาและไม่ใช่เมลานอมา

| โมเดล | ความแม่นยำ | Recall เมลานอมา | F1 เมลานอมา |
|-------|-----------|-----------------|-------------|
| Logistic Regression | ~67-70% | ~0.60-0.65 | ~0.30-0.35 |
| Decision Tree (ความลึก 5) | ~65-68% | ~0.55-0.62 | ~0.28-0.33 |
| Random Forest | ~69-72% | ~0.60-0.65 | ~0.32-0.36 |

*หมายเหตุ: ค่าที่แน่นอนอาจแตกต่างเล็กน้อยในแต่ละครั้งที่รัน เนื่องจาก random seed ค่าที่แสดงเป็นช่วงตัวแทน*

## 6. การอภิปราย

### การตีความทางคลินิก

การวิเคราะห์ยืนยันรูปแบบหลายอย่างที่ได้รับการยอมรับในวรรณกรรมด้านผิวหนัง:

- **อายุเป็นตัวทำนายความรุนแรงที่แข็งแกร่งที่สุดในข้อมูลเมตา** ความเสี่ยงของเมลานอมา, BCC และเกล็ดจากแสงแดดเพิ่มขึ้นอย่างมากตามอายุ สะท้อนทศวรรษของการสัมผัส UV สะสมและการลดลงของความสามารถในการซ่อมแซม DNA ตามอายุ

- **ตำแหน่งบนร่างกายสะท้อนรูปแบบการสัมผัส UV** บริเวณที่สัมผัสแสงแดด (หลัง ใบหน้า หนังศีรษะ) มีอัตรามะเร็งที่เกี่ยวข้องกับ UV สูงกว่าสัดส่วน เมลานอมาที่ลำตัวและหลังเป็นรูปแบบความเสี่ยงสูงที่ทราบแล้ว โดยเฉพาะในผู้ป่วยชาย

- **ความแตกต่างทางเพศในการกระจายแผลมีอยู่จริงแต่ไม่มาก** การพบเนื้องอกไฟโบรมาผิวหนังในหญิงมากกว่าเป็นข้อสังเกตที่มีเอกสารรองรับ ความแตกต่างทางเพศในภาพรวมมีขนาดเล็กกว่าและซับซ้อนกว่าผลของอายุหรือตำแหน่ง

- **ข้อมูลเมตามีสัญญาณคัดกรองที่มีความหมาย** แม้มีเพียงสามตัวแปร โมเดลก็ให้ recall ของเมลานอมาในช่วง 60-65% แม้ยังไม่เพียงพอสำหรับการวินิจฉัยทางคลินิก แต่แสดงว่าการคัดกรองตามกฎง่าย ๆ มีพื้นฐานทางคลินิก

### ข้อพิจารณาเรื่องความไม่สมดุลของชั้น

การครอบงำของ `nv` ในชุดข้อมูล (67%) มีผลกระทบสำคัญ:

- การทดสอบทางสถิติอาจถูกขับเคลื่อนโดยชั้นส่วนใหญ่
- โมเดล ML ที่ฝึกโดยไม่จัดสมดุลจะทำนายไม่ใช่เมลานอมาเป็นค่าเริ่มต้นและได้ความแม่นยำสูงเทียม
- Recall สำหรับชั้นส่วนน้อย (เมลานอมา) เป็นตัวชี้วัดประสิทธิภาพที่มีความหมาย

การใช้ `class_weight='balanced'` ช่วยแก้ปัญหานี้บางส่วน แต่การใช้งานจริงจะต้องใช้วิธีเพิ่มเติม เช่น oversampling (SMOTE) หรือการปรับ threshold

## 7. ข้อจำกัด

1. **ความไม่สมดุลของชุดข้อมูล:** ไฝเม็ดสีคิดเป็น 67% ของชุดข้อมูล ชั้นที่หายาก เช่น เนื้องอกไฟโบรมาผิวหนังและแผลหลอดเลือด มีขนาดตัวอย่างจำกัด ทำให้การประมาณทางสถิติมีความน่าเชื่อถือน้อยลง

2. **การวิเคราะห์เฉพาะข้อมูลเมตา:** โครงการนี้ตั้งใจไม่ใช้ข้อมูลภาพ คุณลักษณะภาพ (สี เนื้อผิว ขอบไม่สม่ำเสมอ) เป็นสัญญาณวินิจฉัยหลักในไดเออร์โมสโกปี รูปแบบจากข้อมูลเมตาเป็นเพียงข้อมูลเสริม

3. **อคติของชุดข้อมูลที่คัดเลือก:** HAM10000 รวบรวมในสถานการณ์วิจัยด้านผิวหนังเฉพาะทาง ไม่แทนประชากรแผลทั้งหมดที่พบในการดูแลปฐมภูมิ ฉุกเฉิน หรือ telemedicine

4. **การเติมค่าอายุ:** เติมค่าอายุที่ขาดด้วยมัธยฐานตามกลุ่ม ทำให้เกิดอคติเล็กน้อย โดยเฉพาะในชั้นที่มีขนาดตัวอย่างเล็ก

5. **หลายภาพต่อหนึ่งแผล:** แผลบางรายการปรากฏเป็นหลายภาพในชุดข้อมูล การวิเคราะห์ที่นี่อยู่ในระดับภาพ ไม่ใช่ระดับแผล

6. **ใช้เพื่อวัตถุประสงค์วิชาการเท่านั้น:** ผลลัพธ์และโมเดลทั้งหมดสร้างขึ้นเพื่อการศึกษา และยังไม่ได้รับการตรวจสอบสำหรับการใช้งานทางคลินิก

## 8. สรุป

โครงการนี้สำรวจรูปแบบทางคลินิกและประชากรศาสตร์ในชุดข้อมูลแผลผิวหนัง HAM10000 โดยใช้การวิเคราะห์เฉพาะข้อมูลเมตา การทดสอบสมมติฐานทางสถิติ และโมเดลการเรียนรู้ของเครื่องแบบง่าย

ผลการค้นพบหลัก:

- **อายุ เพศ และตำแหน่งบนร่างกายของผู้ป่วย มีความสัมพันธ์อย่างมีนัยสำคัญกับชนิดการวินิจฉัยแผล** (Chi-square: p < 0.001; Kruskal-Wallis: p < 0.001)
- **ผู้ป่วยเมลานอมามีอายุมากกว่าอย่างมีนัยสำคัญ** กว่าผู้ป่วยที่ไม่ใช่เมลานอมา (Mann-Whitney U: p < 0.001) โดยช่วงอายุ 41-80 มีสัดส่วนคล้ายร้ายแรงสูงสุด
- **ตำแหน่งบนร่างกายบางแห่งทำนายชนิดแผลได้อย่างชัดเจน:** ใบหน้าและหนังศีรษะสำหรับ BCC และเกล็ดจากแสงแดด แขนขาส่วนล่างสำหรับเนื้องอกไฟโบรมาผิวหนัง และหลัง/ลำตัวสำหรับเมลานอมา
- **โมเดล ML เฉพาะข้อมูลเมตาสามารถให้ recall ของเมลานอมาได้ประมาณ 60-65%** จากเพียงสามคุณลักษณะ แสดงศักยภาพคัดกรองที่แท้จริงแต่จำกัด
- ชุดข้อมูลมี **ความไม่สมดุลของชั้น** อย่างมีนัยสำคัญ (nv = 67%) ที่ต้องจัดการในการใช้งานโมเดลทุกครั้ง

ข้อมูลเมตาทางคลินิกจับรูปแบบที่มีความหมายทางชีววิทยาในข้อมูลแผลผิวหนัง แม้ไม่สามารถแทนการวิเคราะห์ภาพไดเออร์โมสโกปีหรือการตรวจทางคลินิกได้ การเข้าใจรูปแบบเหล่านี้มีคุณค่าในการสร้างกรอบการจัดชั้นความเสี่ยง การคัดกรองการส่งต่อ และการออกแบบโปรแกรมคัดกรองที่ดีขึ้น งานในอนาคตควรรวมคุณลักษณะภาพกับข้อมูลเมตาเพื่อให้ได้โมเดลที่สมบูรณ์และนำไปใช้ทางคลินิกได้มากขึ้น
